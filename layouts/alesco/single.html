{{ i18n "About ALESCo" }}

{{ i18n "Dubbed “air traffic control” for all engineering matters, ALESCo will ensure AlmaLinux’s robustness, reliability, and sustainability while working collaboratively with and overseeing technical decisions in relevant Special Interest Groups (SIGs). The committee will hold five key responsibilities:" }}

  • {{ i18n "Technical oversight for the operating system" }}
  • {{ i18n "Transparency in engineering decisions" }}
  • {{ i18n "Ensuring Long-Term Stability Focus" }}
  • {{ i18n "Release Management" }}
  • {{ i18n "SIG support" }}

{{ i18n "ALESCo brings together the maintainers, packagers, and SIG leads who build and sustain AlmaLinux OS. It is not a closed body. Anyone working on AlmaLinux is welcome to bring proposals, raise concerns, and help shape technical direction in the open." }}

{{ i18n "How ALESCo works" }}

{{ i18n "ALESCo meets every two weeks, and the schedule is posted on events.almalinux.org. Decisions are made by majority vote of the committee, and the chair role rotates among members every six months. Discussion happens in the open in the alesco channel on chat.almalinux.org, and technical proposals are tracked as pull requests in the ALESCo repository on GitHub. Anyone working on AlmaLinux is welcome to join a meeting, raise a topic, or open a proposal." }}

{{ i18n "Lifecycle of a proposal" }}

{{ i18n "Proposals to ALESCo are written as RFCs (requests for comments) and submitted as pull requests in the ALESCo repository. Each one moves through a clear set of stages, in the open." }}

1 {{ i18n "Submitted" }}

{{ i18n "A new RFC is opened as a pull request and checked for formatting." }}

2 {{ i18n "Open for Discussion" }}

{{ i18n "The committee and community review and refine it in the open. The standard review period is two weeks." }}

3 {{ i18n "Final Comment Period" }}

{{ i18n "ALESCo signals that a decision is close and invites any last feedback." }}

4 {{ i18n "Decision" }}

{{ i18n "ALESCo makes the final call, weighing community feedback. A proposal ends in one of these outcomes:" }}

{{ i18n "Accepted" }} {{ i18n "Rejected" }} {{ i18n "Withdrawn" }}

{{ i18n "In rare cases that touch trademarks, significant resources, or partnerships, the AlmaLinux Foundation Board provides input before a vote." }}
