What Contributing Could Look Like for You or Your Team




Content Creation

Create engaging content showcasing AlmaLinux use in real-world projects



Enhancing ELevate

Help expand ELevate migrations and add PES data for more package support



Improving Our Wiki

Expand, refine, or improve AlmaLinux wiki documentation for the community



Maintaining Our Infrastructure

Help maintain core infrastructure like builds, testing, and mirrors



Keeping Things Secure

Join the security team – manage audits, logs, and define security practices



Building Packages

Rebuild RHEL packages, handle EPEL requests, and support the build process



Serving as an Expert

Offer expert help on our Mailing List, Mattermost chat, and Reddit channels



Working at Events

Staff local events and represent AlmaLinux at LUGs and open-source conferences



Boost the Brand

Contribute with web design, graphics, swag, or updates to almalinux.org



Assist in Our Certification

Support efforts for CIS, STIG, and Common Criteria security certifications



Developing

Build or improve platforms like mirrors, builds, or other tools (Python/JS)



Translating

Help expand and improve AlmaLinux website translations for global users
