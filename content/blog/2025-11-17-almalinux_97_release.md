# AlmaLinux 9.7 Stable Now Available

Hello Community! The AlmaLinux OS Foundation is announcing the general availability of [AlmaLinux OS 9.7](https://mirrors.almalinux.org/isos.html) codenamed "Moss Jungle Cat"!

Installation ISOs are available on the mirrors now for all 4 architectures:

- [Intel/AMD (x86_64)](https://mirrors.almalinux.org/isos/x86_64/9.7.html)
- [ARM64 (aarch64)](https://mirrors.almalinux.org/isos/aarch64/9.7.html)
- [IBM PowerPC (ppc64le)](https://mirrors.almalinux.org/isos/ppc64le/9.7.html)
- [IBM Z (s390x)](https://mirrors.almalinux.org/isos/s390x/9.7.html)

Torrents are available as well at:

- [Intel/AMD (x86_64)](https://repo.almalinux.org/almalinux/9.7/isos/x86_64/AlmaLinux-9.7-x86_64.torrent)
- [ARM64 (aarch64)](https://repo.almalinux.org/almalinux/9.7/isos/aarch64/AlmaLinux-9.7-aarch64.torrent)
- [IBM PowerPC (ppc64le)](https://repo.almalinux.org/almalinux/9.7/isos/ppc64le/AlmaLinux-9.7-ppc64le.torrent)
- [IBM Z (s390x)](https://repo.almalinux.org/almalinux/9.7/isos/s390x/AlmaLinux-9.7-s390x.torrent)

## Release Notes and More Information

AlmaLinux 9.7 introduces performance enhancements, updated development tools, and improved security. This release delivers new compiler toolsets and updated module streams, along with improvements to debugging and networking tools. Container and virtualization support is enhanced with the latest versions of Podman and Buildah. Security is improved with updated SELinux policies, OpenSSL featuring post-quantum cryptography support, and newer versions of SSSD and Keylime.

You can read the full release notes for this version on the wiki: [AlmaLinux OS 9.7 Release Notes](https://wiki.almalinux.org/release-notes/9.7.html).
