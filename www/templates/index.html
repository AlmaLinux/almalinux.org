{% translate 'About AlmaLinux OS' %}

{% blocktranslate trimmed %} AlmaLinux OS is an open-source, community-driven Linux operating system that fills the gap left by the discontinuation of the CentOS Linux stable release. AlmaLinux OS is a 1:1 binary compatible clone of RHEL® guided and built by the community. {% endblocktranslate %}

{% blocktranslate trimmed %} As a standalone, completely free OS, AlmaLinux OS enjoys $1M in annual sponsorship from CloudLinux Inc and support from other sponsors. Ongoing development efforts are governed by the members of the community. {% endblocktranslate %}

{% blocktranslate trimmed %} The AlmaLinux OS Foundation is a 501(c)(6) non-profit created for the benefit of the AlmaLinux OS community. {% endblocktranslate %}
